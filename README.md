papers
==================================================
各種書類。Markdown,CSS形式にてマスターデータを管理。

構造
------
- 1.md   … マスターデータ
- 2.html … 中間ファイル
- 3.pdf  … 最終出力

変換方法
--------
    $ cd directory_to_project
    $ ./update_pdf.sh
